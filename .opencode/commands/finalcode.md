Use the FinalCode skill to audit the current repository.

If no arguments are provided, run Inspect Mode (the default).

If an argument is provided, detect the mode:
- "inspect" → Run FinalCode in Inspect Mode
- "repair" → Run FinalCode in Repair Mode
- "refactor" → Run FinalCode in Refactor Mode
- "certify" → Run FinalCode in Certify Mode

Argument: $ARGUMENTS
